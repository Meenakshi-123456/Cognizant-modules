# Stored Procedures (PL/SQL)

## Objective
Implement stored procedures for banking operations: interest processing, employee bonuses, and fund transfers.

## Scenarios
1. ProcessMonthlyInterest - applies 1% interest to all savings accounts
2. UpdateEmployeeBonus - adds a bonus percentage to salaries in a given department
3. TransferFunds - transfers funds between accounts with balance validation

## Output

Monthly interest of 1% applied to all savings accounts.
10% bonus applied to department: IT
Transferred 500 from Account 1 to Account 2
