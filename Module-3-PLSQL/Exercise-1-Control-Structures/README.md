# Control Structures (PL/SQL)

## Objective
Demonstrate loops and conditional logic in PL/SQL using banking scenarios.

## Scenarios
1. Apply 1% interest discount to loans for customers above 60
2. Promote customers to VIP status if balance exceeds $10,000
3. Send reminders for loans due within the next 30 days

## Output

Discount applied for Customer ID: 1
Discount applied for Customer ID: 3

Customer ID 1 promoted to VIP
Customer ID 3 promoted to VIP

Reminder: Customer Alice (ID: 1) has a loan due on 21-JUL-2026
Reminder: Customer Carol (ID: 3) has a loan due on 31-JUL-2026
