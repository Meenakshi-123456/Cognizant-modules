# Arrange-Act-Assert (AAA) Pattern, Test Fixtures, Setup and Teardown

## Objective
Organize JUnit tests using the AAA pattern, and use @Before/@After for setup and teardown.

## Concepts covered
- Arrange: prepare test data and objects
- Act: perform the operation being tested
- Assert: verify the result matches expectations
- @Before: runs setup logic before each test method
- @After: runs cleanup logic after each test method

## Output (expected)
Setup: Test data initialized.
Teardown: Test data cleared.
Setup: Test data initialized.
Teardown: Test data cleared.
