# Assertions in JUnit

## Objective
Demonstrate different JUnit assertions to validate test results.

## Assertions covered
- assertEquals - checks two values are equal
- assertTrue - checks a condition is true
- assertFalse - checks a condition is false
- assertNull - checks a value is null
- assertNotNull - checks a value is not null

## Output (expected)
All assertions in testAssertions() pass, indicating the method completes without throwing any assertion errors.
